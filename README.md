# Татьяна Лаврентьева — «Код доступа к себе»

Премиальный интерактивный лендинг психолога-консультанта, психосоматолога и тренера женской йоги. Реализован как точная, но усиленная версия оригинального макета: тот же контент и фотографии, но с плавным скроллом, параллаксом, поэтапными раскрытиями и живой дыхательной практикой.

## Технологии

- **Next.js 16** (App Router) + **React 19**
- **TypeScript**
- **Tailwind CSS v4** (дизайн-токены в `oklch`)
- **Motion** (Framer Motion) — анимации, параллакс, счётчики, аккордеон
- **Lenis** — инерционный плавный скролл
- **next/font** — Cormorant Garamond (дисплейный) + Manrope (текст), с кириллицей

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
```

Production:

```bash
npm run build
npm run start
```

## Структура

```
app/
  layout.tsx        # шрифты, метаданные, OG
  page.tsx          # сборка секций
  globals.css       # дизайн-система, палитра, зерно, keyframes
components/
  SmoothScroll.tsx  # Lenis + плавные якоря
  Header.tsx        # липкая шапка, прогресс, активная секция, мобильное меню
  Hero.tsx          # параллакс-герой, маска-раскрытие заголовка
  About.tsx         # портрет + анимированные счётчики (Counter.tsx)
  Practice.tsx      # интерактивный «дыхательный» круг (вдох/выдох)
  Faq.tsx           # аккордеон
  Contact.tsx       # форма с экраном благодарности
  ... и остальные секции
lib/content.ts      # весь текст — дословно с оригинала
public/images/      # hero, portrait, breath, og-cover
```

## WOW-детали

- Маска-раскрытие заголовка героя по словам, параллакс изображения и «печать»-кольцо
- Инерционный скролл (Lenis) + прогресс-бар и подсветка активного раздела
- Магнитные кнопки, золотой отсвет за курсором, тонкое зерно поверх страницы
- Счётчики (15+, 500+, 3), поэтапные reveal-анимации всех секций
- Живая практика «Три вдоха возвращения» — круг дышит, текст «Вдох / Выдох» синхронно
- Тёмный navy-блок отзывов и тёплый градиентный CTA для контраста ритма
- Полная адаптивность и поддержка `prefers-reduced-motion`

## Заметки

- Ссылки на Telegram и VK — заглушки (`https://t.me/`, `https://vk.com/`), как в оригинале; подставьте реальные адреса в `lib/content.ts`.
- Форма показывает экран благодарности на клиенте (бэкенд не подключён) — как в исходном макете. Для реальных заявок добавьте обработчик отправки в `components/Contact.tsx`.
